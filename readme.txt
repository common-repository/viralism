Fetch Viral videos/images and create automated post.

== Description ==

This plugin is used to fetch Youtube Videos, Vimeo Videos, Images and create post to specific category.
Admin can choose categories and Twitter and Pinterest account to share the posts on both the social media account at once.

For more details go through the link http://www.viralsoftwares.com/viralism

== Installation ==

For a manual installation via FTP:

1. Extract the downloaded plugin and upload to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' screen in your WordPress admin area

To upload the plugin through WordPress, instead of FTP:

1. You can also [download](https://downloads.wordpress.org/plugin/viralism.zip "Download Viralism") the plugin.
2. Upload the downloaded zip file on the 'Add New' plugins screen (see the 'Upload' tab) in your WordPress admin area and activate.
3. Activate the plugin through the 'Plugins' screen in your WordPress admin area

For any query or details please visit [Plugin Home](http://www.viralsoftwares.com/viralism)
